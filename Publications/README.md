# Reproducible R code for Ukrainian ESS 2022 publications

| Publication Title | Folder/Script | Description |
|-------------------|---------------|-------------|
| [Social changes in Ukraine and Europe: Evidence from the European Social Survey (rounds 6 and 10)](https://doi.org/10.5281/zenodo.18015364) | `ESS6-10 Tables.R` | Code for constructing and outputting comparison tables between Ukraine and European countries for ESS rounds 6 and 10. |
| [Social changes in Ukraine and Europe: Evidence from the European Social Survey (rounds 10 and 11)](https://zenodo.org/records/18595343) | `ESS610-11 Tables.R` | Code for constructing and outputting comparison tables between Ukraine and European countries for ESS rounds 10 and 11. |
| [War Beyond Borders: How Does It Echo Across Europe?](https://voxukraine.org/en/war-beyond-borders-how-does-it-echo-across-europe) | `War Beyond Borders.R`,<br> `War Beyond Borders_Data.RData` | Code and data for reproducing the analysis presented in the corresponding publication. The dataset includes all necessary countries and variables. The methodology and results are described in the publication. |
| [Linear or U-shaped? Age and subjective well-being in Ukraine](https://stmm.in.ua/archive/ukr/2026-1/9.pdf) | `Paradox of well-being.R` | Code for reproducing the analysis presented in the corresponding article. Includes data download and preparation (construction of the period variable), statistical tests for group comparisons, regression analysis (CCREM models), as well as scripts for generating regression table and figure. |
