# Presentations from Events

This section features presentations delivered at various events, presenting findings from the 6th, 10th, and 11th rounds of the European Social Survey (ESS), as well as additional analyses beyond these rounds and beyond ESS data. The presentations were created using Canva and Quarto in R and include interactive charts and tables.

*The organization of these events was supported by Documenting Ukraine, a project of the Institute for Human Sciences, IWM Vienna.*
